# Solidity Day 3 - Inheritance & Libraries

## Overview
This project demonstrates Solidity inheritance and library usage.

## Inheritance
- Base contract: Animal
- Child contracts: Dog, Cat
- Uses virtual and override

## Library
- MathLib with subtraction and multiplication
- Used inside Calculator contract

## Functions

### Animal.speak()
Base function to be overridden

### Dog.speak()
Returns "Bark"

### Cat.speak()
Returns "Meow"

### Calculator.subtract()
Uses MathLib to subtract numbers

### Calculator.multiplyNumbers()
Uses MathLib to multiply numbers
